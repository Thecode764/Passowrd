# Passowrd
Password For Files While Running This App
## Install Repo
```
git clone https://github.com/Thecode764/Passowrd
```
## Install Pip Packages
```
pip install colorama
```
## Step 1: Add Password To .password.txt
```
cd Password
nano .password.txt
```
## Step 2: Run App
```
cd Password
python3 saver.py
```
## Add New Password
```
cd Passowrd
nano .password.txt
```
